# 类型

Python 中每一个值都有类型，比如整型、布尔型、字符串等等。

Python 是弱类型语言，变量在声明时不需要指定类型，且类型随时可变。

## 内置类型

Python 提供了许多内置类型，这里列举一些：

| 类型名称    | 类型含义 |
|---------|------|
| `int`   | 整数   |
| `float` | 小数   |
| `bool`  | 布尔值  |
| `str`   | 字符串  |
| `list`  | 列表   |
| `set`   | 集合   |
| `dict`  | 字典   |
| `tuple` | 元组   |

使用 `type()` 函数可以获取一个变量的类型：

```python sandbox
# 使用 type() 获取变量类型
print(type(1), type('Hello, world!'), type(False), type([1, 2, 3]), type((1, 2, 3)))
```

你可以使用 `isinstance()` 函数来判断一个变量是否属于对应类型：

```python sandbox
# isinstance() 判断类型
print(isinstance(1, int), isinstance([1, 2], tuple))
```

类型名本身也是一个函数，可以用来类型转换：

```python sandbox
# 类型转换
print(int('123') + int('456'))
print(str(False) + str(True))
print(bool('True') and bool('False'))
print(list('Hello, world!'))
print(tuple([1, 2, 3]))
```

当然不是任何两个类型都可以转换的，这时候会报错。

## 常量表达式

我们之前讲过的 `'Hello, world!'`、`[1, 2, 3]`、`1, 2, 3`、`True` 这些都属于常量表达式。

简单来说，这些值在程序运行前就已经确定了。

现在我们来学习更多的常量表达式。

### 浮点数

`float` 类型是小数，更专业的名称是浮点数。

最常用的常量表达式就是数学的表达，例如 `3.14`。

当然，你也可以使用科学计数法：`7.2e10`（甚至可以 `72e9`）。

### 整数

我们通常都是用十进制来表示整数的，这时候最高位一定不是 `0`。

如果要用其他进制的话，就需要用最高位为 `0` 来通知程序，此时第二位描述具体使用哪个进制。

例如 `0b???` 是二进制、`0o???` 是八进制、`0x???` 是十六进制。

这里 `b` 表示 binary、`o` 表示 octal、`x` 表示 hex。

在使用 `int()` 从字符串转化为整数时，可以指定第二个参数表示使用的进制，例如：

```python
print(int('123', 8))    # 83
```

### 空容器

一般来说，空容器可以用原本的括号中间不加任何东西表示，例如 `()`、`[]`。

不过比较特殊的是集合和字典的括号是一样的，Python 规定 `{}{}` 表示空字典。

事实上，你可以直接使用类型函数不加任何参数来获得空容器，例如 `set()`。

### 字典

`dict()` 函数事实上可以接受任何参数，并转换为相应的字典：

```python sandbox
d = dict(x=1, y=2, abc=3)
print(d)
```

由于可以省略键的双引号，很多时候这样的写法更简洁。
