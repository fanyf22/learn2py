# 搭建 Python 环境

Python 作为常用编程语言，其环境有多种安装方法。

这里推荐大家直接前往官网下载安装包进行安装。

## 安装 Python

<Steps>

  ### 下载安装包

  首先进入 Python 官网 [https://python.org]()，点开 `Downloads`，下载最新版 Python 安装包。

  <Image name="Python Official Website" />

  ### 运行安装包

  运行下载好的安装包，按照引导一步一步安装。

  <Image name="Python Installer" width="75%" />

  ### 启动 IDLE

  安装完成后，电脑上应该会出现一个名为 IDLE 的应用，启动这个应用。

  你可以在设置里选择一款喜欢的编程字体，个人推荐 Consolas。

  <Image name="Python IDLE" width="75%" />

  ### 添加环境变量（非必须）

  环境变量是系统级别的，所有应用都可以访问环境变量。

  `PATH` 环境变量用于告诉程序启动时在哪里寻找可执行文件，因此将 Python 安装的位置添加到 `PATH` 可以让运行 Python 更加方便。

  一般来说，Windows 安装包会有“将 Python 添加到 PATH”这个选项，在安装时勾选即可；Mac 安装包则会自动将 Python 添加到环境变量，不需要进行额外的操作。

</Steps>

## 测试一下

在 IDLE 中输入 `print('Hello World'){:python}` 并按下回车，看看会有什么反应吧。
