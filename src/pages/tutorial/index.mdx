# Learn2Py

Learn2Py 是一个面向 Python 零基础的教程，旨在教会完全不懂 Python 甚至编程的读者最基本的 Python 编程能力，并培养独立编写项目的能力。

<Callout>
  本教程使用的 Python 版本为 `3.12.3{}`，版本号向下兼容，所以读者安装 Python3 最新版本即可。
</Callout>

## 快速开始

在左侧目录点击任何你感兴趣的模块开始学习。

如果您完全零基础，建议你按目录顺序进行学习。
